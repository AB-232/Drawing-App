# 🎨 Doby Brush

An interactive and feature-rich Android drawing application built with Kotlin and XML, designed to provide a smooth digital canvas experience with clean code practices.

## 📱 Features

* **Custom Canvas View:** Enjoy a smooth free-hand drawing experience with support for multiple strokes.
* **Undo / Redo Functionality:** Easily undo your mistakes or redo paths step-by-step.
* **Brush Size Customization:** Control the brush thickness dynamically using a dedicated SeekBar dialog.
* **Advanced Color Picker:** Choose any custom color grade using an advanced Color Picker dialog alongside the quick-access color palette.
* **Draw on Images:** Import any picture from your device storage to use as a background and draw directly over it.
* **Save Drawings:** Export and save your final artwork directly to your device gallery with a confirmation toast.

## 🛠 Technologies Used

* Kotlin
* Android Studio
* XML Layouts

## ⚠️ Compatibility Note

> [!IMPORTANT]
> **Gallery Import Feature:** The application currently supports importing images seamlessly on **Android 12 (API 32)** and below. 
> Due to the latest privacy updates and media permission changes (`READ_MEDIA_IMAGES`) introduced in **Android 13 (API 33) and above**, image selection may face temporary restrictions on newer devices. A compatibility update for Android 13+ permissions is actively under development for the next release.

## 🎮 App Flow

1. Start with a blank canvas or import a background image from your gallery.
2. Customize your brush size via the brush tool dialog.
3. Select your drawing color from the palette or the advanced color picker.
4. Draw, sketch, and use Undo/Redo to refine your artwork.
5. Click the save icon to export your final drawing directly to your device.
